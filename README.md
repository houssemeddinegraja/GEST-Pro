# GEST Pro - Global Equipement Sanitaires et Thermiques

This repository contains the source code for the official website of **GEST Pro** (Global Equipement Sanitaires et Thermiques Pro), a professional provider of plumbing, heating, and cooling solutions based in Sousse, Tunisia.

## About the Project

The goal of this project is to develop a modern, responsive, and high-performance web presence for GEST Pro to showcase its range of sanitary and thermal equipment. The website serves as a digital storefront for professionals and individuals in the construction and renovation sectors.

## Tech Stack

This project is built using a clean, progressive approach with standard web technologies:

* **HTML5:** Semantic structure and content.
* **CSS3:** Custom styling and responsive layouts (Mobile-first).
* **JavaScript (ES6+):** Interactive components and dynamic UI elements.
